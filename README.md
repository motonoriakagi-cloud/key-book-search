## 全文検索

PDFのテキストをSQLiteインデックスに構築して、ページ単位で全文検索できます。

### セットアップ

```bash
# 依存ライブラリ
pip install PyMuPDF

# インデックス構築（初回のみ、数分かかります）
python3 build_index.py
```

### 使い方

```bash
# キーワード検索
python3 search.py 気胸

# 書籍を絞り込んで検索
python3 search.py 乳癌 --book 乳房Key

# 表示件数を変更
python3 search.py 骨折 --limit 20
```

### 出力例


## 備考

- PDFファイルは容量が大きいため（合計約3GB）、このリポジトリにはインデックス情報のみを管理しています。
- `index.db`（SQLiteインデックス、約99MB）もローカル生成のため除外しています。
